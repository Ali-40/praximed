# 30-Day Pilot Offer

**Sprint 21 / Module 161**
**Date:** 2026-07-09

---

## Who This Is For

Austrian Wahlärzte and private clinics in Vienna.

Typical profile:
- 1–3 doctors
- 1–2 receptionists
- Existing phone line
- Occasional or frequent missed calls
- No current AI receptionist

---

## What Is Included in the Pilot

- AI receptionist demo setup (staging/synthetic first)
- Callback queue dashboard (`/dashboard`)
- Simple clinic settings (name, doctor, hours, language, tone)
- Staff walkthrough — 30–45 minutes
- Synthetic/staging safety setup (no real patient data until clinic is ready)
- Optional staged live phone demo during the walkthrough

---

## What Is NOT Included in the Pilot

The following are outside the pilot scope and must not be promised:

- Production PHI processing (requires separate data protection review)
- Real medical advice or clinical decision support
- Appointment auto-confirmation (staff always confirms)
- Diagnosis, triage, or emergency medical handling
- DSGVO/AVV compliance certification
- Integration with existing clinic management software
- Production compliance certification
- Full multi-language rollout beyond German

These can be discussed after the pilot phase and workflow validation.

---

## Pilot Structure

**Week 1:** Setup and walkthrough
- Configure clinic name, hours, tone in Einstellungen
- Set up staging/demo mode
- Staff walkthrough of dashboard
- Live phone demo with synthetic data

**Week 2–3:** Controlled live trial (optional, if clinic is ready)
- Gradual introduction with real missed calls if AVV/data review completed
- Daily review of callback queue with staff

**Week 4:** Review and decision
- Review callback requests captured during pilot
- Discuss workflow fit and any changes
- Decision: continue / adjust / cancel
