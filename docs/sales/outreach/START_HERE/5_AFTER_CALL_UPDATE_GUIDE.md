# After-Call Update Guide

**After every call, update the master tracker immediately.**

Master tracker location: `docs/sales/outreach/praxisplan_all_high_potential_leads.xlsx`
(Or open `4_MASTER_TRACKER.xlsx` in this folder — same file.)

---

## Fields to update after every call

| Field | What to enter |
|---|---|
| **Outreach Status** | Choose from dropdown — see list below |
| **Call Attempt 1 Date** | Today's date (YYYY-MM-DD) |
| **Call Attempt 1 Result** | What happened — choose from dropdown |
| **Contact Person** | Name of the person you spoke with (e.g. "Petra, Rezeption") |
| **Follow-up Date** | If you need to call back, set the date |
| **Demo Offered** | Yes or No |
| **Demo Booked** | Yes or No |
| **Objection** | What they said (e.g. "Wir haben schon LATIDO") |
| **Notes** | Anything useful — short and professional |
| **Next Action** | Call back / Send email / Schedule demo / Nothing |
| **Last Updated** | Today's date |

---

## Outreach Status — when to use what

| Status | When |
|---|---|
| Not contacted | Default — not yet called |
| Called | You got through to someone |
| No answer | Phone rang, nobody answered |
| Reception reached | Spoke to the receptionist |
| Asked to send email | They said: "schicken Sie uns eine E-Mail" |
| Email sent | You sent the email |
| Follow-up needed | You need to call again |
| Demo offered | You offered a demo in the conversation |
| Demo booked | They said yes to a demo |
| Interested | They showed interest, no demo yet |
| Not interested | They said no clearly |
| Wrong target | Not the right type of practice |
| **Do not contact** | They said stop — mark this and never call again |

---

## Common scenarios

**They say "schicken Sie uns eine E-Mail":**
1. Set status: `Asked to send email`
2. Note the email address in Notes
3. Go to `3_EMAIL_DRAFTS.md`, copy the draft, send manually
4. Set status: `Email sent`

**They are interested:**
1. Set status: `Interested`
2. Set `Demo Offered = Yes`
3. Set Follow-up Date (call back to book demo)

**They book a demo:**
1. Set status: `Demo booked`
2. Set `Demo Booked = Yes`
3. Set Demo Date

**They say no / do not call again:**
1. Set status: `Do not contact`
2. Do not contact them again — ever

---

## Important rules

- Update the tracker **immediately after each call** — not at the end of the day.
- Never record patient names, health data, or any medical information here.
- This is a B2B sales tracker only. No patient data. No PHI.
