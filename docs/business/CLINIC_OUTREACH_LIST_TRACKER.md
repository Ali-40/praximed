# PraxisMed — Clinic Outreach List Tracker

**Sprint:** Sprint 18 / Module 128
**Date:** 2026-07-05
**Geography:** Vienna (Wien) — first outreach market
**Status:** Tracker foundation ready — research and outreach can begin immediately

---

## 1. Purpose

This tracker is the operational companion to the 30-Day Pilot Offer Pack (Module 127).
Use it to log, track, and follow up on every private clinic contact in the Vienna
market. The goal is 50 clinics researched, 5 contacted per day minimum, and at least
1 pilot clinic onboarded within 30 days.

All data collected here must be from **public sources only** (clinic websites, Google
Maps, medical directories, public LinkedIn profiles). No data brokers. No scrapers.
No patient data.

---

## 2. How to Use the Tracker

1. **Research phase (daily):** Add 10 new clinics per day from public sources.
   Fill in: clinic name, specialty, website, public email/phone, address, doctor name
   (from website), fit score, and pain point guess.

2. **Outreach phase (daily):** Contact a minimum of 5 clinics per day.
   Update: contact method used, outreach status, last contacted date.

3. **Follow-up rule:** Follow up exactly 3 business days after first contact if no reply.
   Log the follow-up in "Next follow-up" and update status when done.

4. **Demo tracking:** When a demo is booked, set Demo booked? = Yes and add time/date
   in Notes. After the call, update status to Demo completed and record outcome.

5. **Pilot tracking:** If the clinic is interested in the pilot, set status to
   Pilot interested and begin onboarding steps per the Pilot Pack.

---

## 3. Outreach Status Stages

Use exactly these stage names in the tracker table:

| Stage | Meaning |
|---|---|
| **Not contacted** | In the list but outreach not yet started |
| **Email sent** | Initial email sent; awaiting reply |
| **Called** | Phone call attempted (reached or left message) |
| **Replied** | Clinic has replied by any channel |
| **Demo booked** | 15-minute demo call scheduled |
| **Demo completed** | Demo call happened |
| **Pilot interested** | Clinic wants to proceed with 30-day pilot |
| **Not interested** | Clinic declined; note reason |
| **Follow-up later** | Not right timing; agreed to re-engage later |

## 4. Priority / Fit Scoring

Assign one letter score per clinic:

| Score | Criteria |
|---|---|
| **A** | Strong fit — private specialty practice, likely high call volume, clear missed-call problem, receptive to tech |
| **B** | Possible fit — private practice but uncertain call volume or less clear pain point |
| **C** | Weak fit — large group practice, unclear if private, or has signalled low interest in tech |

Focus Week 1 outreach on all **A** clinics first, then **B**. Skip **C** until A+B are exhausted.

## 8. Safe Claim Reminder

Every outreach message and every demo call must respect these boundaries:

| Rule | Required |
|---|---|
| Show fake-data staging demo only | All demos use synthetic data; no real patient records shown |
| No real patient data | Never use or reference actual patient information |
| No production PHI | Production PHI readiness: NO-GO until C3–C8 hardening complete |
| No DSGVO/production readiness claim | Do not claim DSGVO compliance or production security certification |
| No fabricated testimonials | Do not invent clinic quotes, case studies, or customer numbers |
| No auto-confirmation claim | Always clarify that staff Confirm is required — no automated decisions |
| No medical advice claim | Summaries are factual intake briefs; no diagnosis; no treatment advice |

Use the scripts and objection responses from Module 127
(`docs/business/CLINIC_OUTREACH_30_DAY_PILOT_PACK.md`) verbatim.

## 9. Suggested Research Sources for Vienna Clinics

For populating the 50 rows, use these **public** sources only:

- **WKO Firmen A–Z** (wko.at) — Austrian business registry, searchable by specialty and district
- **Google Maps** — search "Privatarzt Wien [Specialty]"; clinic websites linked
- **docfinder.at** — public Austrian doctor directory with specialty and contact info
- **herold.at** — Austrian business directory
- **Clinic websites directly** — contact page, Impressum (legally required in AT)
- **LinkedIn** — doctor's public profile if they have one
- **ärzteliste.at** — Austrian medical council public directory

**Do not use:** data brokers, scraped datasets, aggregators that combine non-public info,
or any source that requires login to access contact details.
